# Ionic 2 Login Form

Example design layout/form of mobile application with ionic 2 & Bootstrap

1. clone this repository: 
```
git clone https://github.com/marcoamodio/Ionic-login-form.git
```

2. navigate ed enter into:
```
cd Ionic-login-form
```

3. Install dependencies:
```
npm install
```

4. Run:
```
ionic serve 
```

Enjoy!
